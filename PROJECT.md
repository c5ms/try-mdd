# 项目定位

> 最后更新：2026-04-21

---

## 项目信息

- **项目名称**：服装外贸公司ERP系统
- **项目定位**：企业级
- **业务领域**：服装外贸、工贸一体化

---

## 项目背景

服装外贸公司，没有自己的产品和工厂，专门接国外客户的大货订单定制化生产，是工贸一体化的贸易公司。

---

## 核心需求

全公司所有业务线支持，包括信息/资金/物流。

---

## 参与角色

| 角色 | 视角 | 背景 |
|------|------|------|
| 业务经理 | 业务流程、订单管理 | 熟悉外贸业务全流程 |
| 财务主管 | 资金流转、成本核算 | 关注财务合规与资金安全 |
| 物流专员 | 物流跟踪、仓储管理 | 关注货物流转效率 |
| 生产跟单员 | 生产进度、质量控制 | 跟踪工厂生产状态 |
| 系统架构师 | 系统边界、集成方案 | 关注系统可扩展性 |

---

## 设计准则

| 准则 | 内容 |
|------|------|
| 数据一致性 | 聚合内强一致，聚合间最终一致 |
| 并发处理 | 关键操作需要并发控制（订单、库存、财务） |
| 权限控制 | 基于角色的权限控制（RBAC） |
| 业务合规 | 操作日志记录、审批流程 |
| 多币种 | 支持多币种交易与结算 |
| 多汇率 | 实时/历史汇率管理，支持汇率波动核算 |

---

## 核心领域

| 领域 | 类型 | 说明 |
|------|------|------|
| 订单管理 | 核心域 | 客户订单全生命周期管理 |
| 生产管理 | 核心域 | 生产计划、进度跟踪、质量控制 |
| 采购管理 | 支撑域 | 面辅料采购、供应商管理 |
| 库存管理 | 支撑域 | 仓储、出入库管理 |
| 财务管理 | 支撑域 | 收付款、成本核算、多币种结算 |
| 客户管理 | 通用域 | 客户信息、信用管理 |
| 基础数据 | 通用域 | 汇率、币种、组织架构等 |
