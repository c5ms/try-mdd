# 服装外贸公司ERP系统

## 项目简介

服装外贸公司ERP系统，支持工贸一体化贸易公司全业务线管理，包括订单、生产、采购、库存、财务、客户等核心业务领域。

## 核心领域

| 领域 | 类型 | 说明 |
|------|------|------|
| 订单管理 | 核心域 | 客户订单全生命周期管理 |
| 生产管理 | 核心域 | 生产计划、进度跟踪、质量控制 |
| 采购管理 | 支撑域 | 面辅料采购、供应商管理 |
| 库存管理 | 支撑域 | 仓储、出入库管理 |
| 财务管理 | 支撑域 | 收付款、成本核算、多币种结算 |
| 客户管理 | 通用域 | 客户信息、信用管理 |
| 基础数据 | 通用域 | 汇率、币种、组织架构等 |

## 快速开始

使用 `/m-design {领域}` 开始设计领域模型。

### 可用领域

- `/m-design 订单管理` - 设计订单管理领域模型
- `/m-design 生产管理` - 设计生产管理领域模型
- `/m-design 采购管理` - 设计采购管理领域模型
- `/m-design 库存管理` - 设计库存管理领域模型
- `/m-design 财务管理` - 设计财务管理领域模型
- `/m-design 客户管理` - 设计客户管理领域模型
- `/m-design 基础数据` - 设计基础数据领域模型
