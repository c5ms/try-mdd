### 禁止值对象

**强制约束**：值对象改为结构化属性

| 约束 | 说明 |
|------|------|
| 禁止值对象 | 领域模型文件不允许包含"值对象"章节 |
| 禁止值对象定义 | 不允许设计独立的值对象实体 |
| 替代方案 | 结构化属性：`address: object（结构化属性：street、city、...）` |

| 原因 | 说明 |
|------|------|
| 过度DDD抽象 | 值对象是DDD概念，对大模型推演价值低 |
| 降低推演效率 | 大模型推演不需要区分实体/值对象 |
| 污染模型文档 | 值对象章节会污染"大模型内存" |
